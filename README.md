# CFE Exam Mastery

**The smarter way to prepare for the Certified Fraud Examiner (CFE) exam.**

A focused study tool built for CA professionals and fraud investigators preparing for the ACFE CFE certification. Covers all four CFE domains with practice questions, spaced repetition, and performance tracking — at a fraction of the cost of official ACFE materials.

---

## CFE Exam Domains Covered

| Domain | Topics |
|--------|--------|
| Financial Transactions & Fraud Schemes | Asset misappropriation, financial statement fraud |
| Law | Criminal law, civil law, evidence |
| Investigation | Interviewing, sources of information, report writing |
| Fraud Prevention & Deterrence | Ethics, corporate governance, fraud risk |

---

## Features

- Practice questions across all four CFE domains
- Performance tracking and weak-area identification
- Offline-capable PWA (Progressive Web App)
- Clean, distraction-free study interface

---

## Tech Stack

- **React 18** + TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **Supabase** for progress sync
- **PWA** support via vite-plugin-pwa

---

